1. Copy the code below and save it as events-widget.html.

2. Host the file securely (HTTPS) on your web server or GitHub Pages.

3. Embed it anywhere (SharePoint, Canvas, Blackboard, TargetConnect) using an iframe:

<iframe src="YOUR_HOSTED_URL" width="100%" height="450" frameborder="0"></iframe>
