# 🛠️ TargetConnect Community Tools

Welcome to the **TargetConnect Community Tools** repository. This is an open-source collection of widgets, builders, and customization tools designed to help University Careers and Employability teams get the most out of the TargetConnect platform.

These tools were built to solve common challenges: ensuring consistent branding, personalizing student content without complex coding, and extracting data from RSS feeds into modern, responsive layouts.

## 🧰 The Toolkit

### 1. Universal Custom Pane Maker
Generate responsive, targeted HTML banners using Apache Velocity logic. 
* **Features:** Multi-segment audience targeting, 5 design templates (Modern, Notice, Gradient, etc.), and built-in accessibility contrast checking.
* **Safety:** Generates hardened, table-based HTML to ensure layouts don't break in the TargetConnect dashboard.

### 2. Interactive Event Calendar
A beautiful, monthly calendar view that bridges the gap between TargetConnect and student portals.
* **Functionality:** Automatically populates itself by reading your institution's Events RSS feed.
* **User Experience:** Features slick pop-up modals for event details and direct booking links.

### 3. Event HTML Builder
Ensures your internal event descriptions always look premium and professional.
* **Functionality:** Turns raw text into highly styled, branded HTML "pills" and formatted blocks ready to paste into the platform.

### 4. Events & Vacancies RSS Widgets
Plug-and-play list and grid widgets that translate standard RSS feeds into modern web components.
* **Features:** Responsive design, dynamic deadline tracking, and easy iframe embedding.

---

## 📖 How to Use

1.  Visit the **[Live Hub](https://mh-ous.github.io/Target-Connect-Tools/)**.
2.  Select the tool you need.
3.  For builders: Configure your settings and click **Copy Code**.
4.  For widgets: Provide your specific TargetConnect RSS URL in the source code.
5.  Paste the output into a **Custom Pane** or **Event Description** in TargetConnect.
